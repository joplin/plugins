# Joplin Plugin

This is your new Joplin plugin. It is suggested that you use this README file to document your plugin.

For information on how to build or publish the plugin, please see [GENERATOR_DOC.md](./GENERATOR_DOC.md)
Steps to Run

npm install
npm run dist
A dist file will be generated in the publish folder.
Open Joplin → Settings → Plugins → Install from file
Select the .jpl file
Restart Joplin, and the plugin will be active.
